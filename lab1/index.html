Lazada Thailand

4 ai) Interaction Design

  • ชอบที่ hover แล้วมี drop-down ลงมาให้เลือก
  • แต่บางครั้ง hover แล้ว drop-down ลงมาลยก็ทำให้ใช้งานยากตอนจะกด advertisement ข้างล่างแล้วเผลอไปโดนเข้า (ถ้าเป็นตัวเองจะใช้ click แทน hover)
  • Search ชื่อสินค้าแล้วมีตัวเลือก auto-complete ให้อย่างเช่น search ชื่อ brand สินค้าแล้วมีตัวเลือกมาให้

    • ii) Information Architect

  • Categories มี dropdown ชัดเจนทำให้ไปหาข้อมูลตามหมวดหมู่ง่ายขึ้น
  • มีหมวดหมู่ actions ที่นิยมให้เลือกข้างบนชัดเจน ทำให้ง่ายต่อการใช้งาน เช่นถ้าจะ topup สามารถไปกดที่ topup ได้เลย
  • Search bar อยู่ข้างบนสามารถเข้าไปหาสินค้าได้ง่าย
  • Track, login, signup อยู่บนขวา เป็นหมวดหมู่ที่เจาะจงที่ตัว user โดยตรงที่ต้องการลงชื่อเข้าใช้เพื่อที่ e.g. ไป track การจัดส่งหรือไปดูข้อมูลการสั่งซื้อ ทำให้สะดวกต่อการใช้งานเพราะอยู่ในกลุ่มเดียวกัน

    • iii) Scenarios

  • ต้องการหาอาหารหมา Categories มี dropdown ชัดเจนทำให้ตรงไปหมวดหมู่ pets ได้เลย
  • อยากเข้ามาทำ mobile topup เฉยๆ (ไม่สนใจจะสั่งสินค้า) มีหมวดหมู่ actions ที่นิยมให้เลือกข้างบนชัดเจน ทำให้ง่ายต่อการใช้งาน เช่นถ้าจะ topup สามารถไปกดที่ topup ได้เลย
  • อยากเข้ามาหาซื้อโฟมล้างหน้าของ merci => Search bar อยู่ข้างบนสามารถเข้าไปหาสินค้าได้ง่าย และมี autofill ทำให้ง่ายต่อการหา (filter สิ่งที่ไม่สนใจ ออกไป)
  • อยากรู้ว่าขั้นตอนการจัดส่งสินค้าไปถึงไหนแล้ว -> Track, login, signup อยู่บนขวา เป็นหมวดหมู่ที่เจาะจงที่ตัว user โดยตรงที่ต้องการลงชื่อเข้าใช้เพื่อที่ไป track การจัดส่งหรือไปดูข้อมูลการสั่งซื้อ ทำให้สะดวกต่อการใช้งานเพราะอยู่ในกลุ่มเดียวกัน

    • bi) Color

  • Header มีสองส่วน บนสุดที่เกี่ยวกับ user จะใช้สีขาวเทาๆ เพื่อไม่ดึงดูดความสนใจ เพราะเป็นเพียงข้อมูล user เท่านั้น
  • ปุ่ม search มีสีส้ม ดึงดูดความสนใจได้ดี ทำให้อยากกด click เข้าไป
  • สี background ของตัว main content เป็นสี แดง แต่ content ข้างในเป็น grid สีขาวทำให้ content ดูโดดเด่นถ้าเทียบกับสีพื้นหลัง แต่ก็ทำให้เว็ปดูแคป + ไม่สบายตาถ้าเทียบกับเว็ปอื่นเช่น watsons
  • ราคามี font สีแดง ทำให้รู้สึกเหมือนมีอะไรพิเศษ เช่น ลดราคา ราคาเดิมเป็นสีเทามีขีดทำให้ความสนใจอยู่ที่ราคาที่ลดแล้ว

    • ii) Graphic Design

  • Banner ใหญ่เต็มจอ ดึงดูดความสนใจไปที่ promotion ใหม่
    • ทกให้อยากกดเข้าไปดู
  • มี section vouchers ที่ตกแต่งสวยงาม เป็นรูปบัตรเหมือนตั๋วหนัง
  • ช่วง get cashback ทำออกมาคล้ายปุ่มแต่ใหญ่มากและเป็นสีฟ้า เข้ากับพ่้นหลังได้ดี

    • iii) Layout

  • Banner ใหญ่เต็มจอ ดึงดูดความสนใจไปที่ promotion ใหม่ทำให้อยากกดเข้าไปดู
  • section deals ที่ขาย product ทำออกมาเป็น card เล็กๆแต่ละ card มีข้อมูลครบและจัดวางได้เท่ากันเป็น grid ทำให้ดูสบายตาน่าใช้
  • ทุกๆ section จะมี margin เท่ากันหมดทำให้ดูสะอาด เรียบร้อย
  • Search bar ข้างบน ไม่มีกรอบ ทำให้ข้อมูลข้างบนไม่ดูรกเกินไป

    • iV) Font

  • font ข้างบนจะเท่ากันถ้าอยู่กลุ่มเดียวกัน ทำให้ดูเป็นระเบียบเรียบร้อย
  • ใน items card -> font ราคา จะใหญ่กว่า font อื่นๆ เพื่อดึงความสนใจ
  • font ช่วง promotions จะใหญ่มากๆ เพื่อแสดงถึงความสำคัญ (ทำให้ดูตื่นเต้น)
