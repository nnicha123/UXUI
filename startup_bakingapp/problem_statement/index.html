Exploration - Problem Statement

When I sign in to the website. I want to be able to click remember-me so I don't have to resign-in every-time

  • Idea 1 = QR Code sign-in
  • Idea 2 = Remember me button

When signed in to the website. I want to have an option to be directed to my dashboard so I don't have to search for it

  • Idea 1 = Button at the top to redirect to dashboard
  • Idea 2 = Sign-in automatically leads to dashboard

When I return to the site after I've been signed out, I want to be notified of any changes so I can take quick action

  • Idea 1 = Notifications (highlighted) like facebook to show unseen actions
  • Idea 2 = Sign-in leads to dashboard showing changes (rating,followers)
  • Idea 3 (wild) = Activity logger of users shown to us every-time changes occur

When in the main page I want to be able to filter for my work using my name so users can find works by the same authors easily

  • Idea 1 = Filter button (search by author category)
  • Idea 2 (wild) = Delete all other users' works leaving only yours
  • Idea 3 = Group by author name and put the most rated one at the top (to be found more easily)

When in another page I want to have a return button/method so I can return to the main page easily

  • Idea 1 = Add a return button to re-direct
  • Idea 2 = Always re-direct back to your page after a set time e.g.30 min
  • Idea 3 = Refresh and return to the home page

When in my page I want users to be able to learn about me (about me section) so hopefully companies can find my profile

  • Idea 1 = Add blogger biography on their dashboard page
  • Idea 2 = Automatically redirect to their social media page so users can learn more about them
  • Idea 3 = Users sent emails introducing bloggers and their works

When in my page I want users to be able to be linked to my social media (instagram/blog) so they can follow my instagram/blog and I could receive more recognition

  • Idea 1 = Icons at the bottom of each work to link to social medias
  • Idea 2 = Users automatically directed to author's social media page when click to view game
  • Idea 3 = Users sent emails of social media information of bloggers

Whilst trying out other's games I want to be able to comment/give advices/rate to other posters and they do the same for me so we could help each other improve/suggest ideas

  • Idea 1 = Icons to rate, give off stars at the bottom of page
  • Idea 2 = Give links to their social media so bloggers can contact each other
  • Idea 3 = Link to outlook so they can email each other giving feedback
